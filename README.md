### 介绍

模仿手机微信端的 `羊了个羊` 游戏
![game|200](./screenshots//game.png)

### 功能

- [x] 单个图层，生成左右对称的图形排列
- [x] 单卡片位置，偏移量随机
- [x] 多图层叠加时，图层可见性的判断
- [ ] 选择卡片后，卡片加入底部候选区的路径动画
- [ ] 候选区卡片满足消除条件时，消失动画
